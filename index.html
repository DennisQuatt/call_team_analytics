Methodology: Calls pulled live from the Aircall API (authoritative; matched to agents on email). Deal metrics from Snowflake DIM_DEALS (Consumer Orders pipeline, real deals only). "Created" counts deals whose creation date falls in the window; "signed"/"converted"/"lost" count deals whose respective event date falls in the window. Meetings = first sales appointments with APPOINTMENT_STATUS = 'Completed', bucketed by DATE_APPOINTMENT_FIRST (canceled / no-shows excluded). Chill pre-order haircut: any deal whose deal name contains "Chill Pre-Order" counts at 30% of its full AMOUNT in every revenue figure (created / sent / signed / converted / MTD). Deal counts are unaffected.
